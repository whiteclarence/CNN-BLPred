Runs in Python 2.7

Required Packages:
1. Propy (https://code.google.com/archive/p/protpy/)
2. Cython (pip install Cython)
3. Cogent (pip install cogent)
4. Pandas (pip install pandas)
5. Numpy (pip install numpy)
6. Tensorflow (pip install tensorflow)
7. TFLearn (pip install tflearn)

Example:

python cnn-blpred.py -i <inputfile> -o <outputfile>
